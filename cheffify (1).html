API-sleutel instellen

Cheffify heeft een Anthropic API-sleutel nodig om recepten te genereren. De sleutel wordt veilig lokaal opgeslagen op jouw toestel.

📋 Hoe krijg ik een sleutel? 1. Ga naar console.anthropic.com
2. Maak een gratis account aan
3. Klik op API Keys → Create Key
4. Kopieer de sleutel en plak hem hieronder
Nieuwe accounts krijgen gratis tegoed om te starten.
